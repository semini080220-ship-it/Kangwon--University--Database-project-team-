# 🎤 발표자 시연 설명서

> 발표 맡은 팀원이 **자기 Windows 컴퓨터에서** 시연 프로그램·발표 대본·PPT를 모두 준비해
> 발표하는 방법을 처음부터 끝까지 정리한 안내서입니다. (개발 지식 없어도 그대로 따라 하면 됩니다.)

---

## 0. 시작하기 전에 (체크리스트)

- [ ] **Windows 노트북/PC** (64-bit) — 이 시연 프로그램은 Windows 전용입니다.
- [ ] **인터넷** — 파일을 *처음 내려받을 때만* 필요. 다운로드만 끝내면 발표장은 오프라인이어도 시연됩니다.
- [ ] **여유 공간 약 200MB**
- [ ] (있으면 좋음) **Microsoft PowerPoint** — 발표 대본을 화면에 띄워 보려면. 없어도 대안이 있습니다(3번 참고).
- [ ] **발표 10분 전, 미리 한 번 실행해 보기** ← 가장 중요!

받을 파일은 **3개**입니다:

|---|------|--------|------|

| 3 | `삼척_분산관광_발표_노트.pdf` | 레포 첫 화면 | 발표 대본 |

## 1. 파일 내려받기

### 1-1. 시연 프로그램(.exe) — Releases에서
1. 레포 첫 화면 오른쪽의 **`Releases`** 클릭 → **`삼척 분산 관광 데모 v1.0.0`** 클릭
   (바로가기: 위 레포 주소 뒤에 `/releases/latest` 를 붙여도 됩니다.)
2. 맨 아래 **`Assets`** 에서 **`ecotour-demo-win-x64.zip`** 클릭 → 다운로드
3. 받은 zip을 **마우스 오른쪽 → "압축 풀기"** (또는 더블클릭 후 "모두 압축 풀기")
   → `EcotourDemo` 라는 폴더가 생깁니다.

### 1-2. 발표 슬라이드(PPT)
- 레포 첫 화면에서 **`삼척_분산관광_발표.pptx`** 클릭 → 오른쪽 **`Download`**(↓) 버튼

### 1-3. 발표 대본(PDF)
- 레포 첫 화면에서 **`삼척_분산관광_발표_노트.pdf`** 클릭 → **`Download`**

> 💡 셋을 한 번에: 레포 첫 화면 초록 **`Code`** 버튼 → **`Download ZIP`** 을 누르면
> PPT·대본이 한꺼번에 받아집니다. **단, 시연 프로그램(.exe)은 용량이 커서 Releases에만** 있으니
> 1-1은 따로 받아야 합니다.

## 2. 시연 프로그램 실행하기

> "설치"가 따로 없습니다. **압축만 풀면 끝**이고, 관리자 권한도 필요 없습니다.

1. `EcotourDemo` 폴더를 엽니다.
2. **`EcotourDemo.exe`** 를 **더블클릭**합니다.
3. **파란 SmartScreen 경고**가 뜨면 → **`추가 정보`** → **`실행`**
   (우리 팀이 만든 프로그램이라 안전합니다. 서명이 없어서 뜨는 경고일 뿐입니다.)
4. **검은 콘솔 창**이 하나 뜹니다. → **이 창이 서버입니다. 끄지 마세요.**
5. 몇 초 뒤 **기본 브라우저에 데모 화면이 자동으로 열립니다** → `http://localhost:8080`
   - 자동으로 안 열리면, 브라우저 주소창에 직접 **`localhost:8080`** 을 입력하세요.
6. **종료할 때**는 그 검은 콘솔 창을 닫으면 됩니다.

✅ 화면에 삼척 관광지 목록·혼잡도·추천·댓글이 보이면 정상입니다.

## 3. 발표 대본 보면서 PPT 발표하기

### 방법 A — PowerPoint가 있을 때 (추천: 청중에겐 안 보이게 대본 보기)
1. `삼척_분산관광_발표.pptx` 를 PowerPoint로 엽니다.
   (대본은 **이미 각 슬라이드 노트에 들어가 있습니다.**)
2. 키보드 **`Alt` + `F5`** → **발표자 보기**가 켜집니다.
   - **빔프로젝터/모니터를 연결**하면: 청중 화면엔 **슬라이드만**, 내 노트북 화면엔 **슬라이드 + 대본**이 나옵니다.
   - 화면 아래 화살표로 다음 슬라이드로 넘깁니다.
3. 연결 화면이 없을 때도 `Alt`+`F5` 로 대본을 보며 연습할 수 있습니다.

> PowerPoint에서 발표자 보기가 안 켜지면: 상단 **`슬라이드 쇼`** 탭 → **`발표자 도구 사용`** 체크 → **`처음부터`**.

### 방법 B — PowerPoint가 없을 때
- **슬라이드**: 무료 **PowerPoint Online**(office.com) 이나 **구글 슬라이드**(드라이브에 업로드)로 열기.
- **대본**: 받아 둔 **`삼척_분산관광_발표_노트.pdf`** 를 휴대폰·태블릿으로 띄워 두거나 **인쇄**해서 보면 됩니다.
  (이 PDF는 한 페이지에 *슬라이드 그림 + 그 슬라이드 대본*이 같이 있어 그대로 읽으면 됩니다.)

## 4. 발표 당일 추천 순서

1. **발표 직전 준비**
   - `EcotourDemo.exe` 를 실행해 페이지가 뜨는 것까지 확인하고 **켜 둡니다**(콘솔 창 유지).
   - PowerPoint를 `Alt`+`F5` 발표자 보기로 띄웁니다.
2. **슬라이드 진행** — 대본을 보며 1번 슬라이드부터 발표.
3. **실제 시연 타이밍** — **10번(분산 추천 로직)·12번(화면)** 슬라이드쯤에서
   브라우저(이미 켜 둔 `localhost:8080`)로 **전환**해 실제 동작을 보여 줍니다.
   - 관광지 목록 보기 → 혼잡도/로컬 명소 → 추천 → 댓글 작성 등을 직접 눌러 시연.
4. **마무리** — 13번 슬라이드로 끝내고, 끝나면 **콘솔 창을 닫아 종료**.

> 🎯 팁: 발표자 보기(PPT)와 브라우저(시연)를 **`Alt`+`Tab`** 으로 빠르게 오갈 수 있게 미리 연습해 두세요.

## 5. 문제가 생기면 (FAQ)

| 증상 | 해결 |
|------|------|
| 파란 **SmartScreen** 경고 | **추가 정보 → 실행**. 안전합니다(우리 팀 프로그램). |
| **백신/방화벽**이 막음 | "허용" 또는 예외 추가. 방화벽이 물으면 **액세스 허용**. |
| 브라우저가 **자동으로 안 열림** | 직접 주소창에 `localhost:8080` 입력. |
| 페이지가 **안 뜸 / 비어 있음** | 서버 기동에 5~10초 걸립니다. 잠깐 기다렸다 **새로고침(F5)**. |
| `localhost:8080` 이 **이미 사용 중** | 8080을 쓰는 다른 프로그램을 끄고 `EcotourDemo.exe` 재실행. |
| 콘솔 창이 **떴다가 바로 사라짐** | 압축을 **완전히** 풀었는지 확인(폴더 통째로). zip 안에서 바로 실행하면 안 됩니다. |
| 내 컴퓨터가 **Mac/리눅스** | 이 `.exe`는 **Windows 전용**입니다. 6번(소스 실행)을 보거나 Windows PC를 사용하세요. |
| 발표장에 **인터넷이 없음** | 괜찮습니다. **다운로드·압축 풀기만 미리** 해 두면 실행 자체는 **오프라인에서도** 됩니다(내 컴퓨터 안에서 도는 서버라서요). |

## 6. (선택) 개발자용 — 소스로 직접 실행

Java 17 이상이 깔린 PC라면 `.exe` 없이도 실행할 수 있습니다.

```powershell
# 레포 폴더 안에서
.\mvnw.cmd spring-boot:run
# 그 다음 브라우저에서 http://localhost:8080
```

`.exe` 를 다시 만들고 싶다면 README의 **「원클릭 데모 (.exe)」 → "직접 다시 빌드하려면"** 항목을 참고하세요.
